# Nexel

Nexel is a modern web application built with [Next.js](https://nextjs.org) designed to provide a seamless and efficient workspace experience.

## Preview

![Login](./public/Login.png)

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
